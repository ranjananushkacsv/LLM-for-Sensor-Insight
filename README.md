# Smart Thermostat AI Assistant

A powerful web application that combines smart thermostat data analysis with AI-powered recommendations for optimizing HVAC systems and reducing energy costs.

## Features

| Feature | Description |
|---------|-------------|
|  **AI Chat Assistant** | Ask questions about your HVAC system and get personalized, data-driven advice |
|  **Interactive Dashboards** | Visualize temperature patterns, energy consumption, and cost analysis |
|  **Local Processing** | Uses Ollama with Phi model - your data never leaves your computer |
|  **Real-time Analysis** | Get instant insights from your actual thermostat data |
|  **Personalized Tips** | Receive customized recommendations based on your usage patterns |
| **Trend Analysis** | Identify usage patterns and peak consumption hours |
